# Nebula North Star

Nebula is a production-grade, real-time **flat deterministic** visualization app for project/file trees, optimized for seamless comprehension at any scale. The core experience prioritizes **predictability over spectacle**: stable 2D layout, zero-jitter live updates, fast pan/zoom, instant filtering/search, and clear hierarchy/edge readability. Every automated change must improve at least one of these pillars without degrading the others: **reliability** (stable builds, deterministic behavior, safe rollback), **performance** (bounded render cost, low-latency updates), **usability** (legible structure and smooth interaction), and **maintainability** (typed contracts, testable modules, auditable operations).

The hourly RALPH loop must only ship bounded, reversible patches that move the system toward this end-state and leave a traceable validation trail. Use `docs/upgrades/VISUAL_REFERENCE.md` as optional inspiration, but never at the expense of deterministic UX, clarity, and measurable runtime stability.
